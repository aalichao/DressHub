# DressHub
Repo for DressHub

## Core Features (must-have by Dec 4)
	- Search Feature
        - Users can search for specific clothing items (e.g. “black tuxedo”) using keyword search and filters like category and size
	
    - Borrow/Rental Feature
        - Users can select an item, choose rental duration, and add the item to their cart

    - User Profiles
        - Users can define physical attributes such as gender, height, and weight for better clothing matching

    - Return Portal 
	    - Provide a necessary portal for users to facilitate returning a clothing rental

    - Wishlist
	    - Users can save items that they wish to rent later in the future

## Stretch Goals (nice-to-have)
	- Review / Rating System 
    - Seller / Buyer Messaging
    - Scheduling for specific clothes
